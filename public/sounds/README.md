# أصوات الإشعارات

هذا المجلد يحتوي على ملفات الصوت للإشعارات.

## الملفات المدعومة

- `notification.mp3` - ملف صوتي للإشعارات (اختياري)

## كيفية الاستخدام

1. **إضافة ملف صوتي مخصص:**
   - ضع ملف `notification.mp3` في هذا المجلد
   - الصيغ المدعومة: MP3, WAV, OGG
   - الحجم الموصى به: أقل من 100 KB

2. **إذا لم يكن هناك ملف صوتي:**
   - النظام سيستخدم Web Audio API تلقائياً لإنشاء صوت تنبيه
   - الصوت سيكون نغمتين بسيطتين

## متى يتم تشغيل الصوت؟

- عند استقبال إشعار جديد في المتصفح (foreground)
- عند استقبال إشعار في الخلفية (background) عبر Service Worker
- عند اكتشاف إشعارات جديدة عند تحديث العداد في topbar

## ملاحظات

- الصوت يعمل فقط إذا كان المستخدم قد منح صلاحية الإشعارات
- يمكن للمستخدم كتم صوت المتصفح من إعدادات المتصفح
- الصوت الافتراضي (Web Audio API) يعمل بدون ملفات إضافية
